# TabBar 图标说明

## 需要的图标文件

请在此目录下放置以下6个图标文件：

### 1. 项目图标
- `project.png` - 未选中状态（灰色 #999999）
- `project-active.png` - 选中状态（蓝色 #1890FF）

推荐图标：文件夹、看板、项目图标

### 2. 统计图标
- `stats.png` - 未选中状态（灰色 #999999）
- `stats-active.png` - 选中状态（蓝色 #1890FF）

推荐图标：柱状图、数据分析、报表图标

### 3. 我的图标
- `mine.png` - 未选中状态（灰色 #999999）
- `mine-active.png` - 选中状态（蓝色 #1890FF）

推荐图标：用户、个人中心图标

## 图标规格

- 尺寸：81px × 81px（推荐使用 @3x 图）
- 格式：PNG（支持透明背景）
- 颜色：
  - 未选中：#999999
  - 选中：#1890FF（研发部）或 #FF6B35（运营部）

## 图标资源

可以从以下平台获取图标：
- iconfont (https://www.iconfont.cn/)
- iconpark (https://iconpark.oceanengine.com/)
- 自行设计

## 临时方案

在正式图标准备好之前，可以暂时使用文字或uni-app内置图标。
